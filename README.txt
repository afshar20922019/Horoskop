Horoskop API
Ett enkelt REST API för sitt horoskop 
Motivation
Den motivationen och tanken bakom att skapa detta projekt var att bygga ett enkelt REST API för sitt horoskop med kombination av Html, Css (bootstrap) och JavaScript(jquery-ajax).
Horoskop använder:
Horoskop använder ett antal open source-projekt för att fungera korrekt:
•	Bootstrap 
•	Jquery - ett JavaScript-bibliotek som är tänkt att förenkla HTML-, DOM- och CSS-modifikation
•	Ajax- en förkortning som står för Asynchronous JavaScript and XML
